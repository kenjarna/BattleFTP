# UI Commands -- **All Commands Are Lowercase**

## Movement

* Move _keyword_
  * Forwards = **F**
  * Backwards = **B**
  * Left = **L**
  * Right = **R**

## Inventory

* I _keyword_
  * Show = **Show**
* Use _keyword_
  * Heal = **Food**
* Pickup **Action**

## Combat

* Attack: keysmash the enter key

## Game

* Exit
